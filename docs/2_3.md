# Задача 2.3

## Условие

Изучение роста десятилетних мальчиков одной московской школы на основе случайной выборки объёмом 26 мальчиков показало, что их средний рост по выборке составляет 118 см с выборочным среднеквадратическим отклонением 6 см. Найдите 80%-ный доверительный интервал для среднеквадратического отклонения, которое характеризует рост всех десятилетних мальчиков московских школ. Предполагается, что рост мальчиков подчиняется нормальному закону распределения.

---

## 1. Обозначения

- $n = 26$ — объём выборки
- $\bar{x} = 118$ см — выборочное среднее
- $s = 6$ см — выборочное среднеквадратическое отклонение
- $\sigma$ — генеральное среднеквадратическое отклонение (СКО)
- $\gamma = 0.80$ — доверительная вероятность
- $\alpha = 1 - \gamma = 0.20$ — уровень значимости

## 2. Предположения

- Генеральная совокупность распределена нормально: $X \sim \mathcal{N}(\mu, \sigma^2)$
- Выборка случайная и независимая

## 3. Основная формула

При нормальном распределении генеральной совокупности доверительный интервал для дисперсии $\sigma^2$ строится на основе распределения хи-квадрат:

$$P\left(\chi^2_{\alpha/2}(n-1) \le \frac{(n-1)s^2}{\sigma^2} \le \chi^2_{1-\alpha/2}(n-1)\right) = \gamma = 0.80$$

Разрешая неравенство относительно $\sigma$, получаем доверительный интервал для **среднеквадратического отклонения**:

$$\boxed{\sigma \in \left(s\sqrt{\frac{n-1}{\chi^2_{1-\alpha/2}(n-1)}};\; s\sqrt{\frac{n-1}{\chi^2_{\alpha/2}(n-1)}}\right)}$$

**Пояснение:** $\chi^2_{\alpha/2}$ — нижний квантиль (левая критическая точка), $\chi^2_{1-\alpha/2}$ — верхний квантиль (правая критическая точка). Поскольку $\chi^2$ в знаменателе, меньшее значение $\chi^2_{\alpha/2}$ даёт верхнюю границу для $\sigma$, а большее $\chi^2_{1-\alpha/2}$ — нижнюю.

## 4. Расчёты

Число степеней свободы: $\nu = n - 1 = 25$

Уровень значимости: $\alpha = 0.20$, значит $\alpha/2 = 0.10$, $1 - \alpha/2 = 0.90$

Квантили распределения $\chi^2$ с 25 степенями свободы:

- $\chi^2_{0.10}(25) \approx 16.473$ (нижняя критическая точка)
- $\chi^2_{0.90}(25) \approx 34.382$ (верхняя критическая точка)

Выборочная дисперсия: $s^2 = 6^2 = 36$

Нижняя граница для $\sigma$:
$$
\sigma_L = s\sqrt{\frac{n-1}{\chi^2_{0.90}(25)}} = 6 \cdot \sqrt{\frac{25}{34.382}} = 6 \cdot \sqrt{0.7272} = 6 \cdot 0.8528 = 5.117
$$

Верхняя граница для $\sigma$:
$$
\sigma_U = s\sqrt{\frac{n-1}{\chi^2_{0.10}(25)}} = 6 \cdot \sqrt{\frac{25}{16.473}} = 6 \cdot \sqrt{1.5177} = 6 \cdot 1.2319 = 7.392
$$

## 5. Результат

**80%-ный доверительный интервал для среднеквадратического отклонения:**

$$\boxed{\sigma \in (5.12 \text{ см};\; 7.39 \text{ см})}$$

Соответствующий интервал для дисперсии:

$$\boxed{\sigma^2 \in (26.18 \text{ см}^2;\; 54.61 \text{ см}^2)}$$

## 6. Вывод

С доверительной вероятностью 80% можно утверждать, что истинное среднеквадратическое отклонение роста десятилетних мальчиков московских школ находится в интервале от 5.12 см до 7.39 см. Это означает, что разброс роста вокруг среднего значения (118 см) с вероятностью 80% характеризуется СКО в указанных пределах.

---

## Альтернативное решение: асимптотический интервал

Если бы распределение роста не было нормальным, можно было бы использовать асимптотический доверительный интервал для СКО, основанный на нормальной аппроксимации:

$$\sigma \in s \cdot \left(1 \pm z_{1-\alpha/2} \cdot \frac{1}{\sqrt{2n}}\right)$$

где $z_{1-\alpha/2} = z_{0.90} \approx 1.282$ (для $\gamma = 0.80$):

$$\sigma \in 6 \cdot \left(1 \pm 1.282 \cdot \frac{1}{\sqrt{52}}\right) = 6 \cdot (1 \pm 0.1778) = (4.93;\; 7.07)$$

Однако этот интервал менее точен при $n = 26$ и предполагает большую выборку. Точный интервал на основе $\chi^2$ предпочтительнее.

## Проверочный код Java

```java
public class Task2_3 {
    public static void main(String[] args) {
        int n = 26;
        double s = 6;
        double gamma = 0.80;
        double alpha = 0.20;
        int nu = n - 1;  // 25

        // Квантили хи-квадрат из таблиц:
        double chi2Lower = 16.473;  // chi2_{0.10}(25)
        double chi2Upper = 34.382;  // chi2_{0.90}(25)

        // Доверительный интервал для СКО
        double sigmaLower = s * Math.sqrt(nu / chi2Upper);
        double sigmaUpper = s * Math.sqrt(nu / chi2Lower);

        System.out.printf("chi2_{0.10}(25) = %.3f%n", chi2Lower);
        System.out.printf("chi2_{0.90}(25) = %.3f%n", chi2Upper);
        System.out.printf("80%%-ный ДИ для sigma: (%.3f; %.3f)%n", sigmaLower, sigmaUpper);
        System.out.printf("80%%-ный ДИ для sigma^2: (%.2f; %.2f)%n",
                          sigmaLower * sigmaLower, sigmaUpper * sigmaUpper);
    }
}
```

### Важность предположения о нормальности

Критическое замечание: данный метод построения доверительного интервала опирается на предположение о нормальности распределения роста. Если распределение роста отличается от нормального (например, имеет более тяжёлые хвосты), реальная доверительная вероятность может отличаться от номинальных 80%. Для проверки нормальности можно использовать критерии согласия (Шапиро-Уилка, Колмогорова-Смирнова) или визуальный анализ (QQ-plot).
