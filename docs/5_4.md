# Задача 5.4

## Условие

Предположим, что время, потраченное студентами как первого, так и второго потоков для подготовки к экзамену по экономике, имеет нормальное распределение. Случайным образом было выбрано 30 студентов первого потока. Оказалось, что из этих студентов один потратил на подготовку к экзамену 10 часов, двое по 25 часов, четверо по 30 часов, шестнадцать по 40 часов, четверо по 50 часов, двое по 55 часов и один – 70 часов. При 5%-ном уровне значимости проверить гипотезу о том, что стандартное отклонение времени подготовки к экзамену студентов первого потока не превышает 10 часов.

---

## 1. Обозначения

- $n = 30$ — объём выборки
- $X$ — время подготовки к экзамену (часы)
- $\sigma$ — генеральное стандартное отклонение
- $\alpha = 0.05$ — уровень значимости

Данные (сгруппированы):
| $x_i$ (часы) | 10 | 25 | 30 | 40 | 50 | 55 | 70 |
|---|---|---|---|---|---|---|---|
| $n_i$ (частота) | 1 | 2 | 4 | 16 | 4 | 2 | 1 |

## 2. Формулировка гипотез

Основная гипотеза: $H_0: \sigma \le \sigma_0 = 10$ часов

Конкурирующая гипотеза: $H_1: \sigma > 10$ часов

Примечание: запись «не превышает 10» означает $\sigma \le 10$. Однако стандартный подход использует граничное значение $\sigma_0 = 10$; если мы отвергнем $H_0: \sigma \le 10$ при этом граничном значении, то отвергнем и для меньших значений. Таким образом, фактически проверяем:

$H_0: \sigma^2 = \sigma_0^2 = 100$ против $H_1: \sigma^2 > 100$ (правосторонний тест)

## 3. Выбор теста и предположения

**Используемый тест:** Критерий хи-квадрат для дисперсии нормальной выборки.

**Предположения:**
- Время подготовки распределено нормально: $X \sim \mathcal{N}(\mu, \sigma^2)$
- Выборка случайная и независимая

**Тестовая статистика:**
$$\boxed{\chi^2 = \frac{(n-1)s^2}{\sigma_0^2}}$$

При справедливости $H_0$ статистика имеет распределение $\chi^2$ с $n-1$ степенями свободы.

**Критическая область (правосторонняя):**
$$\chi^2 > \chi^2_{1-\alpha}(n-1)$$

где $\chi^2_{1-\alpha}(n-1)$ — верхняя критическая точка $\chi^2$-распределения с $n-1$ степенями свободы.

## 4. Вычисление выборочной дисперсии

### Шаг 1: Выборочное среднее

$$
\bar{x} = \frac{\sum n_i x_i}{n} = \frac{1\cdot 10 + 2\cdot 25 + 4\cdot 30 + 16\cdot 40 + 4\cdot 50 + 2\cdot 55 + 1\cdot 70}{30}
$$

$$
= \frac{10 + 50 + 120 + 640 + 200 + 110 + 70}{30} = \frac{1200}{30} = 40
$$

### Шаг 2: Выборочная дисперсия

Вычисляем $\sum n_i x_i^2$:

$$
\sum n_i x_i^2 = 1 \cdot 100 + 2 \cdot 625 + 4 \cdot 900 + 16 \cdot 1600 + 4 \cdot 2500 + 2 \cdot 3025 + 1 \cdot 4900
$$

$$
= 100 + 1250 + 3600 + 25600 + 10000 + 6050 + 4900 = 51500
$$

Теперь выборочная дисперсия (смещённая, т.е. деление на $n$):

$$
s_n^2 = \frac{\sum n_i x_i^2}{n} - \bar{x}^2 = \frac{51500}{30} - 40^2 = 1716.667 - 1600 = 116.667
$$

Несмещённая выборочная дисперсия ($s^2$, деление на $n-1$):

$$
s^2 = \frac{n}{n-1} \cdot s_n^2 = \frac{30}{29} \cdot 116.667 \approx 120.690
$$

Выборочное СКО: $s = \sqrt{120.690} \approx 10.986$ часов.

### Шаг 3: Вычисление тестовой статистики

$$
\chi^2 = \frac{(n-1)s^2}{\sigma_0^2} = \frac{29 \cdot 120.690}{100} = \frac{3500.0}{100} = 35.00
$$

Проверим через смещённую дисперсию для контроля:
$$
\chi^2 = \frac{n \cdot s_n^2}{\sigma_0^2} = \frac{30 \cdot 116.667}{100} = \frac{3500.0}{100} = 35.00
$$

## 5. Критическая точка и принятие решения

Число степеней свободы: $\nu = n - 1 = 29$

Критическая точка (из таблиц $\chi^2$):
$$\chi^2_{0.95}(29) \approx 42.557$$

Сравнение:
$$\chi^2_{\text{набл}} = 35.00 < 42.557 = \chi^2_{\text{крит}}$$

**Вывод:** Наблюдаемое значение тестовой статистики не попадает в критическую область. **Нет оснований отвергнуть нулевую гипотезу $H_0$.**

## 6. Результат

$$\boxed{\text{Гипотеза } H_0: \sigma \le 10 \text{ часов не отвергается на 5\%-ном уровне значимости}}$$

## 7. Интерпретация

На основании имеющихся данных нельзя утверждать, что стандартное отклонение времени подготовки превышает 10 часов. Иными словами, гипотеза о том, что разброс времени подготовки к экзамену среди студентов первого потока не превышает 10 часов, **согласуется с данными** на уровне значимости 5%.

Примечание: выборочное СКО $s \approx 10.99$ часов превышает 10 часов, но это превышение недостаточно велико, чтобы считать его статистически значимым при $n = 30$.

---

## Альтернативное решение: через p-value

Вместо сравнения с критической точкой можно вычислить p-value — вероятность получить при справедливости $H_0$ значение статистики $\chi^2$, не меньшее наблюдаемого:

$$p\text{-value} = P(\chi^2(29) \ge 35.00)$$

Из таблиц $\chi^2$-распределения:
$$\chi^2_{0.80}(29) \approx 34.746, \quad \chi^2_{0.75}(29) \approx 31.679$$

Интерполируем: поскольку $35.00$ чуть больше $34.746$, то p-value чуть меньше $0.20$.

Точнее (по более подробным таблицам): p-value $\approx 0.21$.

Поскольку $0.21 > \alpha = 0.05$, нулевая гипотеза не отвергается.

## Проверочный код Java

```java
public class Task5_4 {
    public static void main(String[] args) {
        // Данные
        int n = 30;
        double[] data = new double[n];
        int idx = 0;
        data[idx++] = 10;
        for (int i = 0; i < 2; i++) data[idx++] = 25;
        for (int i = 0; i < 4; i++) data[idx++] = 30;
        for (int i = 0; i < 16; i++) data[idx++] = 40;
        for (int i = 0; i < 4; i++) data[idx++] = 50;
        for (int i = 0; i < 2; i++) data[idx++] = 55;
        data[idx++] = 70;

        double sigma0 = 10;
        double alpha = 0.05;

        // Выборочное среднее
        double sum = 0;
        for (double x : data) sum += x;
        double xBar = sum / n;

        // Несмещённая выборочная дисперсия
        double ssq = 0;
        for (double x : data) ssq += (x - xBar) * (x - xBar);
        double s2 = ssq / (n - 1);
        double s = Math.sqrt(s2);

        // Тестовая статистика
        double chi2Stat = (n - 1) * s2 / (sigma0 * sigma0);

        System.out.printf("Среднее: %.2f%n", xBar);
        System.out.printf("Несмещённая дисперсия s^2: %.4f%n", s2);
        System.out.printf("Выборочное СКО s: %.4f%n", s);
        System.out.printf("Хи-квадрат статистика: %.4f%n", chi2Stat);

        // Критическая точка из таблиц
        double chi2Crit = 42.557;  // chi2_{0.95}(29)
        System.out.printf("Критическая точка chi2_{0.95}(29) = %.3f%n", chi2Crit);
        System.out.printf("Решение: %s%n",
                          chi2Stat > chi2Crit ? "Отвергаем H0" : "Нет оснований отвергнуть H0");

        // Аппроксимация p-value через нормальное распределение
        // При больших nu: sqrt(2*chi2) ~ N(sqrt(2*nu-1), 1)
        int nu = n - 1;
        double z = Math.sqrt(2 * chi2Stat) - Math.sqrt(2 * nu - 1);
        double pValueApprox = 1 - cdfNormal(z);
        System.out.printf("p-value (норм. аппрокс.): %.4f%n", pValueApprox);
    }

    static double cdfNormal(double x) {
        return 0.5 * (1 + erf(x / Math.sqrt(2)));
    }

    static double erf(double x) {
        double t = 1.0 / (1.0 + 0.3275911 * Math.abs(x));
        double tau = t * (0.254829592
                        + t * (-0.284496736
                        + t * (1.421413741
                        + t * (-1.453152027
                        + t * 1.061405429))));
        double y = 1 - tau * Math.exp(-x * x);
        return x >= 0 ? y : -y;
    }
}
```
