# Задача 3.4

## Условие

Из 300 опрошенных будущих избирателей 32% поддержало кандидата партии на пост мэра. В предположении, что проведённый опрос может быть описан в рамках схемы испытаний Бернулли, определить с вероятностью 0,95 интервальную оценку для доли жителей, поддерживающих этого кандидата.

---

## 1. Обозначения

- $n = 300$ — объём выборки (число опрошенных)
- $\hat{p} = 0.32$ — выборочная доля поддерживающих
- $p$ — генеральная доля (истинная доля всех жителей, поддерживающих кандидата)
- $\gamma = 0.95$ — доверительная вероятность
- $\alpha = 1 - \gamma = 0.05$ — уровень значимости

## 2. Предположения

- Схема испытаний Бернулли: каждый опрошенный независимо либо поддерживает кандидата (успех, вероятность $p$), либо нет (неудача, вероятность $1-p$)
- Выборка случайная, репрезентативная
- $n$ достаточно велико для применения нормальной аппроксимации ($n \hat{p} = 300 \cdot 0.32 = 96 \ge 5$ и $n(1-\hat{p}) = 300 \cdot 0.68 = 204 \ge 5$ — условие выполняется)

## 3. Основная формула

Доверительный интервал для доли $p$ (двусторонний, асимптотический, основанный на нормальной аппроксимации):

$$\boxed{p \in \left(\hat{p} \pm z_{1-\alpha/2} \cdot \sqrt{\frac{\hat{p}(1-\hat{p})}{n}}\right)}$$

где $z_{1-\alpha/2}$ — квантиль стандартного нормального распределения уровня $1-\alpha/2$.

Для $\gamma = 0.95$: $\alpha = 0.05$, $\alpha/2 = 0.025$, $z_{0.975} = 1.96$.

## 4. Расчёты

Квантиль: $z_{0.975} = 1.96$

Стандартная ошибка доли:
$$
SE = \sqrt{\frac{\hat{p}(1-\hat{p})}{n}} = \sqrt{\frac{0.32 \cdot 0.68}{300}} = \sqrt{\frac{0.2176}{300}} = \sqrt{0.0007253} \approx 0.02693
$$

Предельная ошибка (margin of error):
$$
\Delta = z_{0.975} \cdot SE = 1.96 \cdot 0.02693 \approx 0.05279
$$

Границы доверительного интервала:
$$
p_L = \hat{p} - \Delta = 0.32 - 0.05279 = 0.2672
$$
$$
p_U = \hat{p} + \Delta = 0.32 + 0.05279 = 0.3728
$$

## 5. Результат

**95%-ный доверительный интервал для доли жителей, поддерживающих кандидата:**

$$\boxed{p \in (0.267;\; 0.373)} \quad \text{или в процентах:} \quad \boxed{p \in (26.7\%;\; 37.3\%)}$$

## 6. Вывод

С вероятностью 0.95 можно утверждать, что истинная доля жителей, поддерживающих данного кандидата, находится в пределах от 26.7% до 37.3%. Поскольку весь интервал лежит ниже 50%, с высокой уверенностью можно сказать, что кандидат не имеет поддержки большинства населения. Однако интервал достаточно широк (около 10.6 процентных пунктов), что связано с ограниченным объёмом выборки.

## Альтернативное решение 1: Интервал Вильсона (Wilson score interval)

Классический интервал на основе нормальной аппроксимации имеет недостатки при малых $n$ или при $p$, близких к 0 или 1. Более точным является интервал Вильсона, особенно при умеренных объёмах выборки:

$$\boxed{p \in \frac{\hat{p} + \frac{z^2}{2n} \pm z \sqrt{\frac{\hat{p}(1-\hat{p})}{n} + \frac{z^2}{4n^2}}}{1 + \frac{z^2}{n}}}$$

где $z = z_{0.975} = 1.96$.

Вычислим:

- $z^2 = 3.8416$
- $\frac{z^2}{2n} = \frac{3.8416}{600} = 0.006403$
- Числитель (центр): $0.32 + 0.006403 = 0.326403$
- Поправка к стандартной ошибке: $\frac{z^2}{4n^2} = \frac{3.8416}{360000} \approx 0.00001067$
- $\sqrt{\frac{0.2176}{300} + 0.00001067} = \sqrt{0.0007360} = 0.02713$
- $z \cdot 0.02713 = 1.96 \cdot 0.02713 = 0.05317$
- Знаменатель: $1 + \frac{3.8416}{300} = 1.01281$

Нижняя граница: $\frac{0.326403 - 0.05317}{1.01281} = \frac{0.27323}{1.01281} = 0.2698$

Верхняя граница: $\frac{0.326403 + 0.05317}{1.01281} = \frac{0.37957}{1.01281} = 0.3748$

Интервал Вильсона: $p \in (0.270;\; 0.375)$ — близок к асимптотическому, но чуть уже и сдвинут вверх.

## Альтернативное решение 2: Метод Агрести-Коула (Agresti-Coull)

Другой подход — «добавить» 2 успеха и 2 неудачи:

- $\tilde{n} = n + 4 = 304$
- $\tilde{p} = \frac{n\hat{p} + 2}{n + 4} = \frac{96 + 2}{304} = \frac{98}{304} = 0.3224$

$$
SE = \sqrt{\frac{\tilde{p}(1-\tilde{p})}{\tilde{n}}} = \sqrt{\frac{0.3224 \cdot 0.6776}{304}} = 0.02681
$$

$$
p \in (0.3224 \pm 1.96 \cdot 0.02681) = (0.2698;\; 0.3750)
$$

Результаты всех трёх методов близки.

## Проверочный код Java

```java
public class Task3_4 {
    public static void main(String[] args) {
        int n = 300;
        double pHat = 0.32;
        double gamma = 0.95;
        double z = 1.96;  // z_{0.975}

        // Метод 1: Асимптотический интервал
        double se = Math.sqrt(pHat * (1 - pHat) / n);
        double delta = z * se;
        System.out.printf("Асимптотический: (%.4f; %.4f)%n", pHat - delta, pHat + delta);

        // Метод 2: Интервал Вильсона
        double z2 = z * z;
        double denom = 1 + z2 / n;
        double center = (pHat + z2 / (2 * n)) / denom;
        double halfWidth = z * Math.sqrt(pHat * (1 - pHat) / n + z2 / (4.0 * n * n)) / denom;
        System.out.printf("Вильсон:       (%.4f; %.4f)%n", center - halfWidth, center + halfWidth);

        // Метод 3: Агрести-Коул
        int nTilde = n + 4;
        double pTilde = (n * pHat + 2) / nTilde;
        double seTilde = Math.sqrt(pTilde * (1 - pTilde) / nTilde);
        double deltaTilde = z * seTilde;
        System.out.printf("Агрести-Коул:  (%.4f; %.4f)%n", pTilde - deltaTilde, pTilde + deltaTilde);
    }
}
```
