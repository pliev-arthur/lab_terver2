# Задача 12

## Условие

Разделив исходный набор данных на две выборки, проверить гипотезу об однородности выборок с помощью критерия Манна-Уитни при уровне значимости 0,01, приведя все промежуточные расчеты.

---

## 1. Обозначения

- Исходная выборка объёма $N = 200$ (столбец 5)
- Выборка $X$: $x_1, ..., x_m$ — первые 100 наблюдений ($m = 100$)
- Выборка $Y$: $y_1, ..., y_n$ — последние 100 наблюдений ($n = 100$)
- $H_0$: выборки однородны (извлечены из одной генеральной совокупности)
- $H_1$: выборки неоднородны (различаются по положению)
- $\alpha = 0.01$ — уровень значимости

## 2. Предположения

- Наблюдения независимы как внутри выборок, так и между выборками
- Распределения непрерывны (критерий Манна-Уитни — непараметрический, не требует нормальности)
- Альтернатива — сдвиг: $F_Y(x) = F_X(x - \Delta)$ при $\Delta \neq 0$

## 3. Описание критерия Манна-Уитни

В качестве статистики критерия Манна-Уитни используется случайная величина $U$, равная числу «успехов» в парных сравнениях выборок $X$ и $Y$. Под «успехом» в сравнении элементов $X_i$ и $Y_j$ понимается выполнение неравенства $X_i < Y_j$. Совпадение элементов признаётся за половину «успеха». Всего производится $mn$ парных сравнений:

$$\boxed{U = \sum_{i=1}^{m} \sum_{j=1}^{n} I\{X_i < Y_j\} + \frac{1}{2} \sum_{i=1}^{m} \sum_{j=1}^{n} I\{X_i = Y_j\}}$$

Числовые характеристики статистики $U$:

$$\boxed{M[U] = \frac{mn}{2}, \qquad D[U] = \frac{mn(m+n+1)}{12}}$$

При объёмах выборок $m, n$ таких, что $m + n > 20$, нулевая гипотеза принимается, если

$$\boxed{\frac{|U - mn/2|}{\sqrt{mn(m+n+1)/12}} < x_{1-\alpha/2}[N(0,1)]}$$

где $x_{1-\alpha/2}[N(0,1)]$ — квантиль стандартного нормального распределения.

## 4. Расчёты

### Шаг 0: Описательные статистики выборок

| Характеристика | Выборка $X$ (№1–100) | Выборка $Y$ (№101–200) |
|:---|---:|---:|
| Объём | $m = 100$ | $n = 100$ |
| Среднее | −3.172 | −5.152 |
| Медиана | −2.994 | −5.119 |
| Мин. | −22.398 | −18.368 |
| Макс. | 10.487 | 6.401 |

Наблюдается заметная разница в средних: выборка $X$ в среднем на ≈2 единицы больше выборки $Y$.

### Шаг 1: Вычисление $U$ через парные сравнения

Статистика $U$ вычисляется прямым подсчётом: для каждого элемента $X_i$ подсчитывается число элементов $Y_j$, превышающих $X_i$ («успехи»). При совпадении значений засчитывается пол-успеха.

Автоматизированный подсчёт (Java):

$$U = 3854.5$$

Для проверки: всего парных сравнений $mn = 100 \cdot 100 = 10000$. Значение $U = 3854.5$ меньше $mn/2 = 5000$, что означает, что элементы выборки $X$ в среднем **больше** элементов выборки $Y$ ($X_i < Y_j$ выполняется реже, чем в половине случаев).

### Шаг 2: Числовые характеристики

$$M[U] = \frac{mn}{2} = \frac{100 \cdot 100}{2} = 5000$$

$$D[U] = \frac{mn(m+n+1)}{12} = \frac{100 \cdot 100 \cdot 201}{12} = \frac{2010000}{12} = 167500$$

$$\sigma[U] = \sqrt{167500} \approx 409.27$$

### Шаг 3: Нормированная статистика

$$\frac{|U - mn/2|}{\sqrt{mn(m+n+1)/12}} = \frac{|3854.5 - 5000|}{409.27} = \frac{1145.5}{409.27} \approx 2.799$$

### Шаг 4: Критическое значение и решение

При $\alpha = 0.01$ (двусторонний тест):

$$x_{1-\alpha/2}[N(0,1)] = x_{0.995}[N(0,1)] = 2.576$$

Сравнение:

$$2.799 > 2.576$$

Наблюдаемое значение попадает в критическую область.

## 5. Результат

$$\boxed{\text{Гипотеза } H_0 \text{ об однородности выборок отвергается на 1\%-ном уровне значимости}}$$

## 6. Вывод

Критерий Манна-Уитни выявил статистически значимое различие между первой и второй половинами выборки ($|U - mn/2|/\sigma \approx 2.80$, $p < 0.01$). Средние (−3.17 vs −5.15) и медианы (−2.99 vs −5.12) существенно различаются, причём первая половина выборки систематически больше второй.

Этот результат **согласуется с критерием инверсий** из задачи 8, который также обнаружил восходящий тренд ($Z = +2.99$). Иными словами, в данных присутствует временной тренд: значения во второй половине выборки значимо ниже, чем в первой.

**Примечание:** Обнаруженная неоднородность не противоречит выводу задачи 11 о нормальности распределения: неоднородность может быть связана с временным трендом (сдвигом среднего во времени), а не с отклонением от нормальной формы распределения в каждый момент времени.

---

## Альтернативное решение 1: Разбиение «чётные/нечётные»

Если разбить выборку иначе — по позиции (чётные vs нечётные индексы):

| Характеристика | Нечётные позиции | Чётные позиции |
|:---|---:|---:|
| Объём | 100 | 100 |
| Среднее | −3.950 | −4.374 |
| Медиана | −3.875 | −4.861 |

$$U = 5101.0, \quad \frac{|U - mn/2|}{\sqrt{mn(m+n+1)/12}} = 0.247$$

$0.247 < 2.576$ — **$H_0$ не отвергается**. Выборки однородны.

## Альтернативное решение 2: Случайное разбиение

При случайном разбиении 200 элементов на две группы по 100:

$$U \approx 5180, \quad \frac{|U - mn/2|}{\sqrt{mn(m+n+1)/12}} \approx 0.44$$

$0.44 < 2.576$ — **$H_0$ не отвергается**.

### Сравнение результатов

| Способ разбиения | $\|U - mn/2\|/\sigma$ | Решение |
|:---|:---:|:---|
| Первые 100 vs последние 100 | 2.80 | $H_0$ **отвергается** |
| Чётные vs нечётные позиции | 0.25 | $H_0$ не отвергается |
| Случайное разбиение | 0.44 | $H_0$ не отвергается |

**Вывод:** Неоднородность обнаруживается только при разбиении по времени (первая половина vs вторая). Это указывает на наличие **временнóго тренда** в данных — последовательного изменения среднего значения от начала к концу выборки. При случайном или перемежающемся разбиении тренд «размывается» между группами и не обнаруживается.
