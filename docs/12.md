# Задача 12

## Условие

Разделив исходный набор данных на две выборки, проверить гипотезу об однородности выборок с помощью критерия Манна-Уитни при уровне значимости 0,01, приведя все промежуточные расчеты.

---

## 1. Обозначения

- Исходная выборка объёма $n = 200$ (столбец 5)
- Выборка 1: $x_1, ..., x_{100}$ — первые 100 наблюдений ($n_1 = 100$)
- Выборка 2: $y_1, ..., y_{100}$ — последние 100 наблюдений ($n_2 = 100$)
- $H_0$: выборки однородны (извлечены из одной генеральной совокупности)
- $H_1$: выборки неоднородны (различаются по положению)
- $\alpha = 0.01$ — уровень значимости

## 2. Предположения

- Наблюдения независимы как внутри выборок, так и между выборками
- Распределения непрерывны (критерий Манна-Уитни — непараметрический, не требует нормальности)
- Альтернатива — сдвиг: $F_2(x) = F_1(x - \Delta)$ при $\Delta \neq 0$

## 3. Описание критерия Манна-Уитни

Критерий проверяет гипотезу о том, что две выборки извлечены из одной генеральной совокупности (однородны).

**Алгоритм:**

1. Объединить обе выборки в одну ($N = n_1 + n_2 = 200$)
2. Упорядочить объединённую выборку по возрастанию
3. Присвоить ранги $R_i$ (от 1 до $N$), при совпадениях — средний ранг
4. Вычислить сумму рангов элементов первой выборки: $R_1 = \sum_{i \in \text{выб.1}} R_i$
5. Статистика Манна-Уитни: $U_1 = R_1 - \frac{n_1(n_1+1)}{2}$
6. $U = \min(U_1, U_2)$, где $U_2 = n_1 n_2 - U_1$
7. При $n_1, n_2 \ge 8$ статистика $U$ асимптотически нормальна:

$$Z = \frac{U - E(U)}{\sqrt{Var(U)}} \sim \mathcal{N}(0, 1)$$

где:

$$E(U) = \frac{n_1 n_2}{2}, \quad Var(U) = \frac{n_1 n_2 (n_1 + n_2 + 1)}{12}$$

## 4. Расчёты

### Шаг 0: Описательные статистики выборок

| Характеристика | Выборка 1 (№1-100) | Выборка 2 (№101-200) |

| Медиана | −2.994 | −5.119 |
| Мин. | −22.398 | −18.368 |
| Макс. | 10.487 | 6.401 |

Наблюдается заметная разница в средних: выборка 1 в среднем на ≈2 единицы больше выборки 2.

### Шаг 1: Ранжирование

Объединяем 200 значений, сортируем по возрастанию. Поскольку значения непрерывны (198 уникальных из 200), совпадений практически нет. Там где они есть, используем средний ранг.

**Наблюдение:** В нижней части распределения преобладают элементы выборки 2, в верхней — выборки 1. Это признак неоднородности (различие в положении распределений).

### Шаг 2: Сумма рангов первой выборки

Автоматизированный подсчёт (Python):

$$R_1 = 11195.5$$

### Шаг 3: Статистика Манна-Уитни

$$U_1 = R_1 - \frac{n_1(n_1+1)}{2} = 11195.5 - \frac{100 \cdot 101}{2} = 11195.5 - 5050 = 6145.5$$

$$U_2 = n_1 n_2 - U_1 = 100 \cdot 100 - 6145.5 = 10000 - 6145.5 = 3854.5$$

$$U = \min(U_1, U_2) = 3854.5$$

### Шаг 4: Стандартизация

$$E(U) = \frac{n_1 n_2}{2} = \frac{100 \cdot 100}{2} = 5000$$

$$Var(U) = \frac{n_1 n_2 (n_1 + n_2 + 1)}{12} = \frac{100 \cdot 100 \cdot 201}{12} = \frac{2010000}{12} = 167500$$

$$\sigma_U = \sqrt{167500} \approx 409.27$$

$$Z = \frac{U - E(U)}{\sigma_U} = \frac{3854.5 - 5000}{409.27} = \frac{-1145.5}{409.27} \approx -2.799$$

### Шаг 5: Критическая область

При $\alpha = 0.01$ (двусторонний тест):

$$z_{\text{крит}} = z_{1-\alpha/2} = z_{0.995} = 2.576$$

Критическая область: $|Z| > 2.576$.

### Шаг 6: Сравнение и решение

$$|Z| = 2.799 > 2.576$$

Наблюдаемое значение попадает в критическую область.

## 5. Результат

$$\boxed{\text{Гипотеза } H_0 \text{ об однородности выборок отвергается на 1\%-ном уровне значимости}}$$

## 6. Вывод

Критерий Манна-Уитни выявил статистически значимое различие между первой и второй половинами выборки ($Z = -2.80$, $p < 0.01$). Средние (−3.17 vs −5.15) и медианы (−2.99 vs −5.12) существенно различаются, причём первая половина выборки систематически больше второй.

Этот результат **согласуется с критерием инверсий** из задачи 8, который также обнаружил восходящий тренд ($Z = +2.98$). Иными словами, в данных присутствует временной тренд: значения во второй половине выборки значимо ниже, чем в первой.

**Примечание:** Обнаруженная неоднородность не противоречит выводу задачи 11 о нормальности распределения: неоднородность может быть связана с временным трендом (сдвигом среднего во времени), а не с отклонением от нормальной формы распределения в каждый момент времени.

## Альтернативное решение 1: Разбиение «чётные/нечётные»

|:---|---:|---:|

$$U = 4899.0, \quad Z = -0.247$$

$|Z| = 0.247 < 2.576$ — **$H_0$ не отвергается**. Выборки однородны.

## Альтернативное решение 2: Случайное разбиение

При случайном разбиении 200 элементов на две группы по 100:

$$U \approx 4820, \quad Z \approx -0.44$$

$|Z| = 0.44 < 2.576$ — **$H_0$ не отвергается**.

### Сравнение результатов

| Способ разбиения | $Z$ | Решение |
|:---|:---:|:---|
| Первые 100 vs последние 100 | −2.80 | $H_0$ **отвергается** |
| Чётные vs нечётные позиции | −0.25 | $H_0$ не отвергается |
| Случайное разбиение | −0.44 | $H_0$ не отвергается |

**Вывод:** Неоднородность обнаруживается только при разбиении по времени (первая половина vs вторая). Это указывает на наличие **временнóго тренда** в данных — последовательного изменения среднего значения от начала к концу выборки. При случайном или перемежающемся разбиении тренд «размывается» между группами и не обнаруживается.

## Расчётный код Java

```java
import java.util.ArrayList;
import java.util.Arrays;
import java.util.Collections;
import java.util.Comparator;
import java.util.List;

public class Task12 {
    static class Pair {
        double val;
        int group;  // 1 или 2
        Pair(double v, int g) { val = v; group = g; }
    }

    public static void main(String[] args) throws Exception {
        double[] allData = DataReader.readColumn("data.xlsx", "Лист1", 4);

        // Разбиение: первые 100 vs последние 100
        double[] sample1 = Arrays.copyOfRange(allData, 0, 100);
        double[] sample2 = Arrays.copyOfRange(allData, 100, 200);
        int n1 = sample1.length, n2 = sample2.length;

        // Объединяем выборки
        List<Pair> combined = new ArrayList<>();
        for (double x : sample1) combined.add(new Pair(x, 1));
        for (double y : sample2) combined.add(new Pair(y, 2));
        combined.sort(Comparator.comparingDouble(p -> p.val));

        // Ранги с учётом совпадений
        double[] ranks = new double[combined.size()];
        int i = 0;
        while (i < combined.size()) {
            int j = i;
            while (j < combined.size() && combined.get(j).val == combined.get(i).val)
                j++;
            double avgRank = (i + j + 1) / 2.0;
            for (int t = i; t < j; t++)
                ranks[t] = avgRank;
            i = j;
        }

        // Сумма рангов первой выборки
        double R1 = 0;
        for (int t = 0; t < combined.size(); t++)
            if (combined.get(t).group == 1)
                R1 += ranks[t];

        // Статистика Манна-Уитни
        double U1 = R1 - n1 * (n1 + 1) / 2.0;
        double U2 = (double) n1 * n2 - U1;
        double U = Math.min(U1, U2);

        double EU = n1 * n2 / 2.0;
        double VarU = n1 * n2 * (n1 + n2 + 1) / 12.0;
        double sigmaU = Math.sqrt(VarU);
        double Z = (U - EU) / sigmaU;

        System.out.printf("R1 = %.1f%n", R1);
        System.out.printf("U = %.1f%n", U);
        System.out.printf("E(U) = %.1f, sigma = %.4f%n", EU, sigmaU);
        System.out.printf("Z = %.4f%n", Z);
        System.out.printf("Critical |Z| = %.3f%n", 2.576);
        System.out.printf("Decision: %s%n", Math.abs(Z) > 2.576 ? "REJECT H0" : "Do NOT reject H0");
    }
}
```
