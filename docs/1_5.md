# Задача 1.5

## Условие

Аналитик рынка ценных бумаг оценивает среднюю дневную доходность определённого вида акций. Случайная выборка за 16 дней показала, что средняя доходность по акциям данного типа составляет 8% с выборочным среднеквадратическим отклонением в 4%. Предполагая, что доходность акции подчиняется нормальному закону распределения, определите 90%-ный левосторонний доверительный интервал для средней доходности интересующего аналитика вида акций.

---

## 1. Обозначения

- $n = 16$ — объём выборки
- $\bar{x} = 8\%$ — выборочное среднее
- $s = 4\%$ — выборочное среднеквадратическое отклонение
- $\mu$ — генеральное среднее (истинная средняя доходность)
- $\gamma = 0.90$ — доверительная вероятность (уровень доверия)
- $\alpha = 1 - \gamma = 0.10$ — уровень значимости

## 2. Предположения

- Генеральная совокупность распределена нормально: $X \sim \mathcal{N}(\mu, \sigma^2)$
- Дисперсия $\sigma^2$ неизвестна, используется её выборочная оценка $s^2$
- В силу неизвестной дисперсии применяется распределение Стьюдента

## 3. Основная формула

Для **левостороннего** доверительного интервала с доверительной вероятностью $\gamma$ при неизвестной дисперсии:

$$\mu \in \left(\bar{x} - t_{\gamma}(n-1) \cdot \frac{s}{\sqrt{n}};\; +\infty\right)$$

где $t_{\gamma}(n-1)$ — квантиль распределения Стьюдента уровня $\gamma$ с $n-1$ степенями свободы.

**Пояснение выбора $t_{\gamma}$, а не $t_{1-\alpha/2}$:** Для левостороннего интервала нам нужна нижняя граница. Вероятность того, что $\mu$ не меньше нижней границы, равна $\gamma = 0.90$. Это соответствует вероятности $\alpha = 0.10$ того, что $\mu$ окажется **левее** нижней границы. Поэтому используется односторонний квантиль $t_{1-\alpha}(n-1) = t_{0.90}(n-1)$, а не двусторонний $t_{1-\alpha/2}$.

## 4. Расчёты

Число степеней свободы: $\nu = n - 1 = 15$

Квантиль распределения Стьюдента $t_{0.90}(15)$:
$$
t_{0.90}(15) \approx 1.341
$$

Стандартная ошибка среднего:
$$
\frac{s}{\sqrt{n}} = \frac{4}{\sqrt{16}} = \frac{4}{4} = 1
$$

Нижняя граница доверительного интервала:
$$
\bar{x} - t_{0.90}(15) \cdot \frac{s}{\sqrt{n}} = 8 - 1.341 \cdot 1 = 8 - 1.341 = 6.659
$$

## 5. Результат

**90%-ный левосторонний доверительный интервал для средней доходности:**

$$\boxed{\mu \in (6.66\%;\; +\infty)}$$

## 6. Вывод

С доверительной вероятностью 90% можно утверждать, что истинная средняя дневная доходность акций данного типа составляет не менее 6.66%. Иными словами, аналитик может быть на 90% уверен, что средняя доходность превышает 6.66%.

---

## Альтернативное решение: через точное распределение

Вместо использования табличного значения $t_{0.90}(15) \approx 1.341$ можно вычислить точный квантиль через функцию распределения Стьюдента с помощью библиотеки Apache Commons Math:

```java
import org.apache.commons.math3.distribution.TDistribution;

public class Task1_5_Alternative {
    public static void main(String[] args) {
        TDistribution tDist = new TDistribution(15);
        double tGamma = tDist.inverseCumulativeProbability(0.90);
        System.out.printf("t_{0.90}(15) = %.4f%n", tGamma);
    }
}
```

Однако табличное значение $t_{0.90}(15) = 1.341$ является общепринятым и даёт тот же результат.

**Проверочный код Java:**

```java
public class Task1_5 {
    public static void main(String[] args) {
        // Исходные данные
        int n = 16;
        double xBar = 8;
        double s = 4;
        double gamma = 0.90;

        // Квантиль t-распределения для уровня 0.90 с 15 ст. св.
        // Из таблиц: t_{0.90}(15) = 1.341
        double tGamma = 1.341;

        // Стандартная ошибка
        double se = s / Math.sqrt(n);

        // Нижняя граница
        double lower = xBar - tGamma * se;

        System.out.printf("Стандартная ошибка: %.4f%n", se);
        System.out.printf("t_{0.90}(15) = %.3f%n", tGamma);
        System.out.printf("Нижняя граница: %.4f%n", lower);
        System.out.printf("90%%-ный левосторонний ДИ: (%.4f; +inf)%n", lower);
    }
}
```

### Сравнение с двусторонним интервалом

Если бы требовалось построить **двусторонний** 90%-ный доверительный интервал, формула была бы:
$$\mu \in \left(\bar{x} \pm t_{0.95}(15) \cdot \frac{s}{\sqrt{n}}\right)$$
где $t_{0.95}(15) \approx 1.753$. Тогда:
$$8 \pm 1.753 \cdot 1 = (6.247;\; 9.753)$$

Левосторонний интервал $(6.66; +\infty)$ имеет более высокую нижнюю границу (ближе к среднему), чем левая граница двустороннего интервала (6.25), что логично — вся «неуверенность» в левостороннем интервале сосредоточена на одной стороне, поэтому нижняя граница оказывается ближе к оценке.
