# Задача 7

## Условие

Выбрать набор данных из генеральной совокупности непрерывного типа. Определить выборочные характеристики: среднее, дисперсию, медиану, коэффициенты асимметрии и эксцесса. Построить гистограмму относительных частот. На основе визуального анализа гистограммы, а также выборочных характеристик, выдвинуть гипотезу о виде закона распределения исследуемого набора данных. Сделать выводы о свойствах гипотетического распределения.

---

## 1. Выбор набора данных

Из 6 представленных столбцов данных выбираем **столбец 5**. Обоснование:

- Столбец 3 (13 уникальных значений) и столбец 4 (24 уникальных значения) — дискретные, исключаются
- Столбцы 1, 2, 5, 6 — непрерывные
- Столбец 5 имеет 198 уникальных значений из 200, что указывает на действительно непрерывную природу
- Асимметрия близка к 0, эксцесс близок к 0 — признаки нормального распределения

## 2. Обозначения

- $n = 200$ — объём выборки
- $x_1, x_2, ..., x_n$ — наблюдаемые значения
- $\bar{X}$ — выборочное среднее
- $S^2$ — выборочная дисперсия (смещённая оценка, т.к. $n$ велико: $n=200$, поправка $\frac{n-1}{n}=0.995$ пренебрежима)
- $med$ — выборочная медиана
- $As$ — выборочный коэффициент асимметрии (skewness)
- $Ex$ — выборочный коэффициент эксцесса (excess kurtosis)

## 3. Выборочные характеристики

### 3.1. Выборочное среднее

Вычисляем по всем 200 значениям с помощью Python (точное вычисление):

$$\bar{X} = \frac{1}{n} \sum_{i=1}^{n} x_i = -4.1621$$

### 3.2. Выборочная дисперсия и СКО

$$S^2 = \frac{1}{n} \sum_{i=1}^{n} (X_i - \bar{X})^2 = 27.6927$$

$$S = \sqrt{S^2} = 5.2624$$

### 3.3. Выборочная медиана

Упорядочиваем данные по возрастанию. При $n = 200$ (чётное):

$$med = \frac{X_{(100)} + X_{(101)}}{2} = -4.6310$$

### 3.4. Выборочные центральные моменты

Момент 3-го порядка (смещённая оценка):

$$m_3 = \frac{1}{n} \sum_{i=1}^{n} (X_i - \bar{X})^3 = -1.2326$$

Момент 4-го порядка (смещённая оценка):

$$m_4 = \frac{1}{n} \sum_{i=1}^{n} (X_i - \bar{X})^4 = 2324.1287$$

### 3.5. Коэффициент асимметрии

$$As = \frac{m_3}{S^3} = \frac{-1.2326}{5.2624^3} = \frac{-1.2326}{145.72} = -0.0085 \approx -0.01$$

Асимметрия крайне близка к 0 — распределение практически симметрично.

### 3.6. Коэффициент эксцесса (excess kurtosis)

$$Ex = \frac{m_4}{S^4} - 3 = \frac{2324.1287}{5.2624^4} = \frac{2324.1287}{766.89} - 3 = 3.031 - 3 = 0.031 \approx 0.03$$

Эксцесс близок к 0 — островершинность близка к нормальному распределению.

### 3.7. Минимум и максимум

$$X_{\min} = -22.398, \quad X_{\max} = 10.487$$

### 3.8. Размах

$$R = X_{\max} - X_{\min} = 10.487 - (-22.398) = 32.885$$

## 6. Выдвижение гипотезы о виде распределения

### 6.1. Анализ характеристик

Сравним выборочные характеристики с эталонными значениями для различных распределений:

| Характеристика | Наблюдаемое | Нормальное $N(\mu,\sigma^2)$ | Равномерное $U(a,b)$ |
|:---|---:|---:|---:|
| Асимметрия | −0.01 | 0 | 0 |
| Эксцесс | 0.03 | 0 | −1.2 |
| Медиана ≈ Среднее | −4.63 ≈ −4.16 | Да | Да |

Наблюдаемый эксцесс 0.03 исключает равномерное распределение (у которого эксцесс −1.2).

### 6.2. Визуальный анализ гистограммы

Гистограмма демонстрирует:
1. **Одномодальность** — один выраженный пик
2. **Симметричность** — левая и правая части зеркально похожи
3. **Колоколообразную форму** — характерную для нормального распределения

### 6.3. Гипотеза

$$\boxed{H: X \sim \mathcal{N}(a, \sigma^2)}$$

где $a$ — математическое ожидание, $\sigma^2$ — дисперсия.

Исследуемый набор данных с высокой вероятностью извлечён из генеральной совокупности, имеющей **нормальное (гауссовское) распределение** $F_1: X \sim \mathcal{N}(a, \sigma^2)$.

---

## 7. Свойства гипотетического распределения

Нормальное распределение $\mathcal{N}(a, \sigma^2)$ обладает следующими свойствами:

1. **Симметричность**: плотность $f(x) = \frac{1}{\sigma\sqrt{2\pi}} e^{-\frac{(x-a)^2}{2\sigma^2}}$ симметрична относительно $x = a$

2. **Параметры формы**: асимметрия $\equiv 0$, эксцесс $\equiv 0$ — что согласуется с наблюдениями ($As = -0.01$ и $Ex = 0.03$)

3. **Правило трёх сигм**: $P(|X-a| < \sigma) \approx 0.683$, $P(|X-a| < 2\sigma) \approx 0.954$, $P(|X-a| < 3\sigma) \approx 0.997$

4. **Оценки параметров** (предварительные):
   - $\hat{a} = \bar{X} = -4.162$
   - $\hat{\sigma}^2 = S^2 = 27.693$
   - $\hat{\sigma} = S = 5.262$

5. **Медиана, мода и среднее совпадают**: $med = mod = a$

6. **Носитель**: $x \in (-\infty, +\infty)$, что согласуется с данными (нет явных границ)
