# Задача 6.2

## Условие

В таблице приведены результаты опроса об объёме расходов на мобильную связь:

| Расходы           | 8 у.е. – 15 у.е. | 15 у.е. – 30 у.е. | Более 30 у.е. |
|-------------------|:----------------:|:-----------------:|:-------------:|
| Количество человек | 312              | 486               | 189           |

С помощью критерия $\chi^2$ проверьте гипотезу о том, что распределение всех пользователей мобильных телефонов по трём группам имеет вид 30%–50%–20% соответственно.

---

## 1. Обозначения

- $k = 3$ — число категорий
- $n_i$ — наблюдаемые частоты: $n_1 = 312$, $n_2 = 486$, $n_3 = 189$
- $n = \sum n_i = 312 + 486 + 189 = 987$ — общий объём выборки
- $p_i^{(0)}$ — гипотетические вероятности: $p_1^{(0)} = 0.30$, $p_2^{(0)} = 0.50$, $p_3^{(0)} = 0.20$
- $\alpha = 0.05$ — уровень значимости (стандартный, явно не указан в условии, но применяется)

## 2. Формулировка гипотез

$$H_0: p_1 = 0.30,\; p_2 = 0.50,\; p_3 = 0.20$$
$$H_1: \text{распределение отличается от указанного (хотя бы одна вероятность не равна гипотетической)}$$

Это критерий согласия (goodness-of-fit): проверяется, соответствует ли эмпирическое распределение теоретическому.

## 3. Выбор теста и предположения

**Используемый тест:** Критерий хи-квадрат Пирсона для проверки гипотезы о виде распределения.

**Тестовая статистика:**
$$\boxed{\chi^2 = \sum_{i=1}^{k} \frac{(n_i - n \cdot p_i^{(0)})^2}{n \cdot p_i^{(0)}}}$$

При справедливости $H_0$ и достаточно больших ожидаемых частотах ($n p_i^{(0)} \ge 5$ для всех $i$) статистика имеет асимптотическое распределение $\chi^2$ с $\nu = k - 1$ степенями свободы. (Поправка: параметры не оценивались по выборке, поэтому $\nu = k - 1 = 2$, а не $k - 1 - r$.)

**Предположения:**
- Выборка случайная, независимая
- Наблюдения распределены по $k$ взаимоисключающим категориям
- Ожидаемые частоты достаточно велики (условие выполняется, см. ниже)

**Критическая область (правосторонняя):**
$$\chi^2 > \chi^2_{1-\alpha}(k-1)$$

## 4. Расчёты

### Шаг 1: Ожидаемые частоты

$$E_1 = n \cdot p_1^{(0)} = 987 \cdot 0.30 = 296.1$$
$$E_2 = n \cdot p_2^{(0)} = 987 \cdot 0.50 = 493.5$$
$$E_3 = n \cdot p_3^{(0)} = 987 \cdot 0.20 = 197.4$$

Проверка: $296.1 + 493.5 + 197.4 = 987.0 = n$ ✓

Все $E_i \ge 5$, условие применимости выполняется.

### Шаг 2: Вычисление компонент $\chi^2$

| $i$ | $n_i$ | $E_i$ | $n_i - E_i$ | $(n_i - E_i)^2$ | $\frac{(n_i - E_i)^2}{E_i}$ |
|-----|-------|-------|-------------|-----------------|------------------------------|
| 1 | 312 | 296.1 | +15.9 | 252.81 | 0.8538 |
| 2 | 486 | 493.5 | −7.5 | 56.25 | 0.1140 |
| 3 | 189 | 197.4 | −8.4 | 70.56 | 0.3574 |
| **Σ** | **987** | **987.0** | **0** | — | **1.3252** |

$$\chi^2_{\text{набл}} = 0.8538 + 0.1140 + 0.3574 = 1.3252 \approx 1.33$$

### Шаг 3: Критическая точка

Число степеней свободы: $\nu = k - 1 = 2$

Из таблиц $\chi^2$ при $\alpha = 0.05$:
$$\chi^2_{0.95}(2) = 5.991$$

### Шаг 4: Сравнение и решение

$$\chi^2_{\text{набл}} = 1.33 < 5.991 = \chi^2_{\text{крит}}$$

## 5. Результат

$$\boxed{\text{Гипотеза } H_0 \text{ о распределении 30\%–50\%–20\% не отвергается на 5\%-ном уровне значимости}}$$

## 6. Интерпретация

Наблюдаемые частоты (312, 486, 189) достаточно близки к ожидаемым при гипотетическом распределении 30%–50%–20% (296.1, 493.5, 197.4). Различия между наблюдаемыми и ожидаемыми частотами статистически незначимы: значение $\chi^2 = 1.33$ при 2 степенях свободы соответствует p-value $\approx 0.51$, что значительно превышает 0.05.

Таким образом, данные опроса **согласуются** с гипотезой о том, что все пользователи мобильных телефонов распределены по трём группам расходов в пропорции 30% : 50% : 20%.

---

## Альтернативное решение: критерий отношения правдоподобия ($G$-тест)

Вместо критерия Пирсона можно использовать критерий отношения правдоподобия:

$$G = 2 \sum_{i=1}^{k} n_i \ln\left(\frac{n_i}{E_i}\right)$$

Статистика $G$ также имеет асимптотическое распределение $\chi^2(k-1)$.

Вычислим:

$$G = 2 \left[312 \cdot \ln\left(\frac{312}{296.1}\right) + 486 \cdot \ln\left(\frac{486}{493.5}\right) + 189 \cdot \ln\left(\frac{189}{197.4}\right)\right]$$

$$= 2 \left[312 \cdot \ln(1.05369) + 486 \cdot \ln(0.98480) + 189 \cdot \ln(0.95742)\right]$$

$$= 2 \left[312 \cdot 0.05232 + 486 \cdot (-0.01532) + 189 \cdot (-0.04351)\right]$$

$$= 2 \left[16.323 - 7.446 - 8.224\right] = 2 \cdot 0.653 = 1.306 \approx 1.31$$

$G = 1.31$ очень близко к $\chi^2 = 1.33$, что подтверждает согласованность результатов. Вывод тот же: $H_0$ не отвергается.

## Проверочный код Java

```java
public class Task6_2 {
    public static void main(String[] args) {
        // Наблюдаемые частоты
        int[] nObs = {312, 486, 189};
        double[] pHyp = {0.30, 0.50, 0.20};
        int n = 0;
        for (int ni : nObs) n += ni;

        // Ожидаемые частоты
        double[] nExp = new double[3];
        for (int i = 0; i < 3; i++)
            nExp[i] = n * pHyp[i];

        System.out.print("Ожидаемые частоты: ");
        for (int i = 0; i < 3; i++)
            System.out.printf("%.1f ", nExp[i]);
        System.out.println();

        // Статистика Пирсона
        double chi2 = 0;
        for (int i = 0; i < 3; i++) {
            double diff = nObs[i] - nExp[i];
            chi2 += diff * diff / nExp[i];
        }
        System.out.printf("Хи-квадрат Пирсона: %.4f%n", chi2);

        // G-тест
        double G = 0;
        for (int i = 0; i < 3; i++)
            G += 2 * nObs[i] * Math.log(nObs[i] / nExp[i]);
        System.out.printf("G-статистика: %.4f%n", G);

        // Критическая точка
        double chi2Crit = 5.991;  // chi2_{0.95}(2)
        System.out.printf("Критическая точка chi2_{0.95}(2) = %.3f%n", chi2Crit);

        // Точное p-value: chi2(2) CDF = 1 - e^{-x/2}
        double pValue = Math.exp(-chi2 / 2);
        System.out.printf("p-value (точное для nu=2): %.4f%n", pValue);
        System.out.printf("Решение: %s%n",
                          chi2 > chi2Crit ? "Отвергаем H0" : "Нет оснований отвергнуть H0");
    }
}
```

**Примечание о p-value:** Для $\chi^2$-распределения с $\nu = 2$ степенями свободы функция распределения имеет простой вид: $F(x) = 1 - e^{-x/2}$. Поэтому точное p-value:

$$p = 1 - F(1.325) = e^{-1.325/2} = e^{-0.6625} \approx 0.516$$

Действительно, $p \approx 0.52 > 0.05$, подтверждая, что данные очень хорошо согласуются с $H_0$.
