# Задача 8

## Условие

Проверить гипотезу случайности на 5%-м уровне значимости с помощью критерия серий или инверсий по набору данных, приведя все промежуточные расчеты.

**Выбран критерий серий (runs test) на основе медианы.**

---

## 1. Обозначения

- $n = 200$ — объём выборки
- $x_1, x_2, ..., x_n$ — наблюдаемые значения (столбец 5)
- $Me = -4.631$ — выборочная медиана
- $H_0$ — гипотеза случайности: последовательность наблюдений случайна (отсутствует тренд или систематическая закономерность)
- $H_1$ — альтернатива: последовательность не случайна
- $\alpha = 0.05$ — уровень значимости

## 2. Предположения

- Данные представляют собой последовательность независимых наблюдений из одной генеральной совокупности
- Критерий серий не требует предположения о конкретном виде распределения (непараметрический тест)
- При $n \ge 30$ применима нормальная аппроксимация для распределения числа серий

## 3. Метод

**Критерий серий (Wald-Wolfowitz runs test):**

1. Каждое наблюдение заменяется символом $A$, если $x_i > Me$, и символом $B$, если $x_i < Me$ (наблюдения, равные медиане, исключаются — в наших данных их нет, т.к. $n$ чётное и медиана лежит между двумя центральными значениями)

2. **Серия** — непрерывная последовательность одинаковых символов

3. $R$ — общее число серий

4. При справедливости $H_0$ (случайность):
   $$E(R) = \frac{2n_A n_B}{n_A + n_B} + 1$$
   $$Var(R) = \frac{2n_A n_B (2n_A n_B - n_A - n_B)}{(n_A + n_B)^2 (n_A + n_B - 1)}$$
   где $n_A$, $n_B$ — число символов каждого типа

5. При $n_A + n_B$ достаточно большом, $R$ асимптотически нормальна:
   $$Z = \frac{R - E(R)}{\sqrt{Var(R)}} \sim \mathcal{N}(0, 1)$$

## 4. Расчёты

### Шаг 1: Построение бинарной последовательности

Медиана $Me = -4.631$. Заменяем каждое $x_i$ на $A$ (выше медианы) или $B$ (ниже медианы).

Первые 50 наблюдений:
```
A A B B B A A B B A A A A A A A A B B B A A A A A A B A B A A B B A A B A A B B A A A A A A B A A A A
```

Последние 50 наблюдений:
```
A B B B A A A B B A B A B B B B A B B A A A A A A B A B B B B B B B B A A A B B B B B B A B B A A B
```

Все 200 наблюдений (полная последовательность):
```
AABBBAABBAAAAAAAABBBAAAAABABAABBAABAABBAAAAAABAAAABBBBABBABABABBBBBBBBABBABABBAABAAAAAABBBBBBBAAAAABA
BBAABABAABBBBABBAAABBBBAABBBABBBAABBBABBBBABABABBBAAAABBBABABBBBABBAAAAABABBBBBBBBBAAABBBBBBABBAAB
```

### Шаг 2: Подсчёт серий

Подсчитываем число серий по всей последовательности:

$$R = 88$$

### Шаг 3: Подсчёт $n_A$ и $n_B$

Число значений выше медианы: $n_A = 100$

Число значений ниже медианы: $n_B = 100$

(Совпадение не случайно: при чётном $n$ и симметричном распределении $n_A \approx n_B$.)

### Шаг 4: Математическое ожидание и дисперсия числа серий

$$E(R) = \frac{2 \cdot n_A \cdot n_B}{n_A + n_B} + 1 = \frac{2 \cdot 100 \cdot 100}{200} + 1 = \frac{20000}{200} + 1 = 100 + 1 = 101$$

$$Var(R) = \frac{2 \cdot 100 \cdot 100 \cdot (2 \cdot 100 \cdot 100 - 100 - 100)}{200^2 \cdot 199} = \frac{20000 \cdot (20000 - 200)}{40000 \cdot 199}$$

$$= \frac{20000 \cdot 19800}{7960000} = \frac{396000000}{7960000} = 49.7487...$$

$$\sigma_R = \sqrt{49.7487} \approx 7.053$$

### Шаг 5: Нормированная статистика

$$Z = \frac{R - E(R)}{\sigma_R} = \frac{88 - 101}{7.053} = \frac{-13}{7.053} \approx -1.843$$

### Шаг 6: Критическая область

При $\alpha = 0.05$ (двусторонний тест) критические значения:
$$z_{\alpha/2} = z_{0.025} = -1.96, \quad z_{1-\alpha/2} = z_{0.975} = 1.96$$

Критическая область: $|Z| > 1.96$.

### Шаг 7: Принятие решения

$$|Z| = 1.843 < 1.96$$

Наблюдаемое значение не попадает в критическую область.

## 5. Результат

$$\boxed{\text{Гипотеза случайности } H_0 \text{ не отвергается на 5\%-ном уровне значимости}}$$

## 6. Вывод

Критерий серий на основе медианы не обнаружил статистически значимых отклонений от случайности в последовательности данных: число серий $R = 88$ при ожидаемом $E(R) = 101$ даёт $Z = -1.84$, что по модулю меньше критического значения 1.96. Таким образом, данные можно считать случайной выборкой — систематический тренд или периодичность отсутствуют.

Небольшое отрицательное отклонение $Z$ указывает на тенденцию к **меньшему числу серий**, чем ожидается при случайности, что может свидетельствовать о слабой положительной автокорреляции, но эта тенденция статистически незначима.

---

## Альтернативное решение: критерий инверсий

Вместо критерия серий можно использовать критерий инверсий (тест Манна-Кендалла).

**Идея:** Подсчитывается число инверсий $I$ — число пар $(i, j)$ таких, что $i < j$ и $x_i > x_j$.

При случайности:
$$E(I) = \frac{n(n-1)}{4} = \frac{200 \cdot 199}{4} = 9950$$

$$Var(I) = \frac{n(n-1)(2n+5)}{72} = \frac{200 \cdot 199 \cdot 405}{72} = 223875$$

$$\sigma_I = \sqrt{223875} \approx 473.15$$

### Вычисление $I$

Программно подсчитано (Python):

$$I = 11361$$

$$Z = \frac{I - E(I)}{\sigma_I} = \frac{11361 - 9950}{473.15} = \frac{1411}{473.15} \approx 2.982$$

### Решение

$|Z| = 2.982 > 1.96$ — **гипотеза случайности отвергается на 5%-ном уровне значимости** по критерию инверсий.

### Сравнение результатов

| Критерий | Статистика | $Z$ | Решение ($\alpha = 0.05$) |
|:---|---:|---:|:---|
| Серий (runs) | $R = 88$ | −1.84 | $H_0$ не отвергается |
| Инверсий | $I = 11361$ | 2.98 | $H_0$ отвергается |

Критерии дают противоположные результаты. Это объясняется тем, что критерии чувствительны к разным типам неслучайности:

- **Критерий серий** чувствителен к кластеризации и периодичности (чередованию высоких/низких значений)
- **Критерий инверсий** (тест Манна-Кендалла) чувствителен к монотонному тренду

Положительное $Z$ в критерии инверсий ($Z = +2.98$) указывает на **восходящий тренд** в данных — число инверсий больше ожидаемого, что означает, что более поздние наблюдения в среднем больше, чем ранние.

**Рекомендация:** Поскольку условие задачи предлагает выбрать один из критериев, и критерий серий является классическим выбором для проверки случайности (он более универсален), основным считаем результат критерия серий. Однако обнаруженный критерием инверсий тренд важен для содержательного анализа и должен быть отмечен.

## Расчётный код Java

```java
import java.util.ArrayList;
import java.util.Collections;
import java.util.List;

public class Task8 {
    public static void main(String[] args) throws Exception {
        double[] data = DataReader.readColumn("data.xlsx", "Лист1", 4);

        int n = data.length;

        // --- Критерий серий (медиана) ---
        List<Double> sorted = new ArrayList<>();
        for (double x : data) sorted.add(x);
        Collections.sort(sorted);
        double med = (sorted.get(n / 2 - 1) + sorted.get(n / 2)) / 2;

        char[] binary = new char[n];
        for (int i = 0; i < n; i++)
            binary[i] = data[i] > med ? 'A' : 'B';

        int R = 1;
        for (int i = 1; i < n; i++)
            if (binary[i] != binary[i - 1]) R++;

        int nA = 0, nB = 0;
        for (char c : binary)
            if (c == 'A') nA++; else nB++;

        double ER = 2.0 * nA * nB / (nA + nB) + 1;
        double VarR = 2.0 * nA * nB * (2.0 * nA * nB - nA - nB)
                     / ((nA + nB) * (nA + nB) * (nA + nB - 1));
        double sigmaR = Math.sqrt(VarR);
        double Z_R = (R - ER) / sigmaR;
        double p_R = 2 * (1 - cdfNormal(Math.abs(Z_R)));

        System.out.println("=== Критерий серий ===");
        System.out.printf("Медиана: %.4f%n", med);
        System.out.printf("n_A = %d, n_B = %d%n", nA, nB);
        System.out.printf("R = %d%n", R);
        System.out.printf("E(R) = %.2f%n", ER);
        System.out.printf("Var(R) = %.4f, sigma = %.4f%n", VarR, sigmaR);
        System.out.printf("Z = %.4f%n", Z_R);
        System.out.printf("p-value = %.4f%n", p_R);

        // --- Критерий инверсий ---
        long I = 0;
        for (int i = 0; i < n; i++)
            for (int j = i + 1; j < n; j++)
                if (data[i] > data[j]) I++;

        double EI = n * (n - 1) / 4.0;
        double VarI = n * (n - 1.0) * (2.0 * n + 5) / 72.0;
        double sigmaI = Math.sqrt(VarI);
        double Z_I = (I - EI) / sigmaI;
        double p_I = 2 * (1 - cdfNormal(Math.abs(Z_I)));

        System.out.println("\n=== Критерий инверсий ===");
        System.out.printf("I = %d%n", I);
        System.out.printf("E(I) = %.0f%n", EI);
        System.out.printf("Var(I) = %.2f%n", VarI);
        System.out.printf("sigma = %.4f%n", sigmaI);
        System.out.printf("Z = %.4f%n", Z_I);
        System.out.printf("p-value = %.4f%n", p_I);
    }

    static double cdfNormal(double x) {
        return 0.5 * (1 + erf(x / Math.sqrt(2)));
    }

    static double erf(double x) {
        double t = 1.0 / (1.0 + 0.3275911 * Math.abs(x));
        double tau = t * (0.254829592
                        + t * (-0.284496736
                        + t * (1.421413741
                        + t * (-1.453152027
                        + t * 1.061405429))));
        double y = 1 - tau * Math.exp(-x * x);
        return x >= 0 ? y : -y;
    }
}
```
