# DataReader

Утилитный класс для чтения и выбора данных из xlsx-файлов через Apache POI.

---

## `readColumn` — чтение столбца по индексу

```java
double[] data = DataReader.readColumn("data.xlsx", "Лист1", 4); // столбец E (0-based)
```

| Параметр | Тип | Описание |
|---|---|---|
| `resourcePath` | `String` | Имя файла в classpath (`src/main/resources/`) |
| `sheetName` | `String` | Имя листа xlsx (например, `"Лист1"`) |
| `columnIndex` | `int` | Индекс столбца (0-based: 0=A, 1=B, ..., 5=F) |

Возвращает `double[]` — все числовые значения столбца. Нечисловые ячейки и пустые строки пропускаются.

Файл ищется в classpath: `DataReader.class.getResourceAsStream("/" + resourcePath)`. Для этого файл `data.xlsx` должен лежать в `src/main/resources/`.

---

## `selectColumnForAnalysis` — автоматический выбор столбца

```java
double[] data = DataReader.selectColumnForAnalysis("data.xlsx", "Лист1");
```

Метод сам определяет, какой из столбцов 1–6 подходит для задач 7–12 (раздел полного анализа данных). Алгоритм:

**Шаг 1 — отсев дискретных столбцов.** Для каждого столбца считается число уникальных значений. Столбцы с менее чем 100 уникальными значениями отбрасываются как дискретные (не подходят для непрерывного анализа).

**Шаг 2 — выбор наиболее симметричного.** Для оставшихся непрерывных столбцов вычисляется коэффициент асимметрии. Выбирается столбец с асимметрией, наиболее близкой к 0 — он наиболее похож на нормальное распределение.

Вывод при запуске программы:

```
=== Автоматический выбор столбца ===
  Столбец 1 (n=200): асимметрия = -0.2981
  Столбец 2 (n=200): асимметрия = -0.0124 <-- выбран
  Столбец 5 (n=200): асимметрия = -0.2489
```

**Гарантии:**
- Если непрерывных столбцов нет — выбрасывается исключение
- Если непрерывный столбец один — он выбирается автоматически
