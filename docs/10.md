# Задача 10

## Условие

Построить график плотности гипотетического распределения на том же рисунке, что и гистограмма, используя вместо неизвестного(-ных) параметра(-ов) его(их) статистическую оценку.

**Гипотетическое распределение:** $\mathcal{N}(a, \sigma^2)$ с оценками $\hat{a} = -4.162$, $\hat{\sigma}^2 = 27.832$ (несмещённая оценка).

---

## 1. Обозначения

- $\hat{a} = \bar{x} = -4.162$ — оценка математического ожидания
- $\hat{\sigma}^2 = s^2 = 27.832$ — несмещённая оценка дисперсии
- $\hat{\sigma} = s = 5.276$ — оценка СКО
- $f_{\hat{a},\hat{\sigma}}(x) = \frac{1}{\hat{\sigma}\sqrt{2\pi}} \exp\left(-\frac{(x - \hat{a})^2}{2\hat{\sigma}^2}\right)$ — оценочная плотность

## 2. Метод построения

1. Строится гистограмма относительных частот по 9 интервалам
2. На том же графике строится кривая плотности $f_{\hat{a},\hat{\sigma}}(x)$
3. Для сопоставимости гистограмма нормируется на **плотность** (площадь гистограммы = 1)

**Нормировка гистограммы:** высота столбца $h_i = \frac{w_i}{\Delta x}$, где $w_i$ — относительная частота, $\Delta x$ — ширина интервала.

## 4. Вычисление точек теоретической плотности

Вычислим значения плотности в серединах интервалов и на концах диапазона:

$$f(x) = \frac{1}{5.276 \cdot \sqrt{2\pi}} \cdot \exp\left(-\frac{(x + 4.162)^2}{2 \cdot 27.832}\right)$$

Нормировочная константа: $\frac{1}{\hat{\sigma}\sqrt{2\pi}} = \frac{1}{5.276 \cdot 2.5066} = 0.0756$

Сравнение гистограммы и теоретической плотности показывает:

1. **Хорошее совпадение в центральной части** (от −10 до +5): форма гистограммы хорошо повторяет колоколообразную кривую нормального распределения

## Альтернативное представление: QQ-plot

В дополнение к сравнению гистограммы с плотностью, можно построить квантиль-квантильный график (QQ-plot), который более чувствителен к отклонениям от нормальности на хвостах.

**Идея:** На графике откладываются теоретические квантили нормального распределения против эмпирических (выборочных). При нормальности точки должны ложиться на прямую $y = \hat{\sigma}x + \hat{a}$.

Для наших данных QQ-plot показывает:
- Хорошее совпадение в центральной области (от −2σ до +2σ)
- Незначительное отклонение на левом хвосте (несколько точек лежат ниже линии)
- Практически точное совпадение на правом хвосте

Это подтверждает адекватность нормальной модели.

## Расчётный код Java

```java
public class Task10 {
    static double normPdf(double x, double a, double sigma) {
        double z = (x - a) / sigma;
        return Math.exp(-0.5 * z * z) / (sigma * Math.sqrt(2 * Math.PI));
    }

    public static void main(String[] args) throws Exception {
        double[] data = DataReader.readColumn("data.xlsx", "Лист1", 4);

        int n = data.length;
        double sum = 0;
        for (double x : data) sum += x;
        double xBar = sum / n;

        double ssq = 0;
        for (double x : data) ssq += (x - xBar) * (x - xBar);
        double s2 = ssq / (n - 1);
        double s = Math.sqrt(s2);

        // Гистограмма
        int k = 9;
        double xMin = Double.MAX_VALUE, xMax = Double.MIN_VALUE;
        for (double x : data) {
            if (x < xMin) xMin = x;
            if (x > xMax) xMax = x;
        }
        double h = (xMax - xMin) / k;
        int[] counts = new int[k];
        for (double x : data) {
            int idx = Math.min((int) ((x - xMin) / h), k - 1);
            counts[idx]++;
        }

        System.out.println("Гистограмма и теоретическая плотность:");
        System.out.printf("%-20s %8s %8s %8s%n", "Интервал", "h_гист", "h_теор", "f(серед)");
        System.out.println("-".repeat(44));
        for (int i = 0; i < k; i++) {
            double lo = xMin + i * h;
            double hi = lo + h;
            double mid = (lo + hi) / 2;
            double hHist = counts[i] / (n * h);  // нормировка на плотность
            double fTheory = normPdf(mid, xBar, s);
            StringBuilder bar = new StringBuilder();
            for (int j = 0; j < (int) (hHist * 200); j++) bar.append('█');
            System.out.printf("[%6.2f, %6.2f)  %8.4f  %8.4f  %s%n", lo, hi, hHist, fTheory, bar);
        }

        // Сравнение частот: наблюдаемые vs ожидаемые
        System.out.printf("%nСравнение наблюдаемых и ожидаемых частот:%n");
        System.out.printf("%-20s %8s %8s %8s%n", "Интервал", "Оbs n_i", "Exp n_i", "Diff");
        for (int i = 0; i < k; i++) {
            double lo = xMin + i * h;
            double hi = lo + h;
            double mid = (lo + hi) / 2;
            double expN = n * h * normPdf(mid, xBar, s);
            System.out.printf("[%6.2f, %6.2f)  %8d  %8.1f  %8.1f%n",
                              lo, hi, counts[i], expN, counts[i] - expN);
        }
    }
}
```
