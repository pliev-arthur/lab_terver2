# Задача 4.4

## Условие

Пробег (в 1000 км) автомобиля «Москвич» до капитального ремонта двигателя является нормальной случайной величиной с неизвестным математическим ожиданием $\mu$ и известной дисперсией $\sigma^2$. Какому уровню доверия соответствует интервал $(93.55;\,106.45)$ для $\mu$, если 95%-ный интервал для $\mu$ по этой выборке принимает значение $(94.84;\,105.16)$?

---

## 1. Обозначения

- $X \sim \mathcal{N}(\mu, \sigma^2)$ — пробег до капремонта, $\sigma^2$ известна
- $I_1 = (93.55;\, 106.45)$ — интервал с неизвестной доверительной вероятностью $\gamma_1$
- $I_{0.95} = (94.84;\, 105.16)$ — 95%-ный доверительный интервал
- $\bar{x}$ — выборочное среднее (неизвестно, но может быть вычислено из симметричных интервалов)

## 2. Предположения

- Генеральная совокупность нормальна: $X \sim \mathcal{N}(\mu, \sigma^2)$
- Дисперсия $\sigma^2$ **известна** → используется стандартное нормальное распределение, а не Стьюдента
- Выборка случайная, независимая, одного и того же объёма $n$ (оба интервала построены по одной выборке)

## 3. Основная формула

При известной $\sigma^2$ доверительный интервал для $\mu$ имеет вид:

$$\boxed{\mu \in \left(\bar{x} \pm z_{1-\alpha/2} \cdot \frac{\sigma}{\sqrt{n}}\right)}$$

где $z_{1-\alpha/2}$ — квантиль стандартного нормального распределения.

## 4. Расчёты

### Шаг 1: Определим $\bar{x}$ из симметрии интервалов

Поскольку оба интервала симметричны относительно $\bar{x}$, выборочное среднее — это середина любого из них:

Для $I_{0.95}$:
$$
\bar{x} = \frac{94.84 + 105.16}{2} = \frac{200.00}{2} = 100.00
$$

Проверим для $I_1$:
$$
\bar{x} = \frac{93.55 + 106.45}{2} = \frac{200.00}{2} = 100.00
$$

Оба интервала симметричны относительно $\bar{x} = 100$ тыс. км.

### Шаг 2: Определим полуширину интервалов

Для $I_{0.95}$ (95%-ный):
$$
\Delta_{0.95} = \frac{105.16 - 94.84}{2} = \frac{10.32}{2} = 5.16
$$

Для $I_1$ (искомый):
$$
\Delta_1 = \frac{106.45 - 93.55}{2} = \frac{12.90}{2} = 6.45
$$

### Шаг 3: Связь полуширин

Полуширина интервала $\Delta = z_{1-\alpha/2} \cdot \frac{\sigma}{\sqrt{n}}$.

Величина $\frac{\sigma}{\sqrt{n}}$ одинакова для обоих интервалов (одна и та же выборка, одна и та же $\sigma$). Поэтому:

$$
\frac{\Delta_1}{\Delta_{0.95}} = \frac{z_{1-\alpha_1/2}}{z_{0.975}}
$$

$$
\frac{z_{1-\alpha_1/2}}{1.96} = \frac{6.45}{5.16} = 1.25
$$

Отсюда:
$$
z_{1-\alpha_1/2} = 1.25 \cdot 1.96 = 2.45
$$

### Шаг 4: Определим уровень доверия $\gamma_1$

Нам нужно найти $\gamma_1 = 1 - \alpha_1$ такой, что:

$$\Phi(z_{1-\alpha_1/2}) = \Phi(2.45) = 1 - \frac{\alpha_1}{2}$$

Из таблиц стандартного нормального распределения:
$$\Phi(2.45) \approx 0.9929$$

Следовательно:
$$1 - \frac{\alpha_1}{2} = 0.9929$$
$$\frac{\alpha_1}{2} = 0.0071$$
$$\alpha_1 = 0.0142$$
$$\gamma_1 = 1 - \alpha_1 = 0.9858 \approx 0.986$$

## 5. Результат

Интервал $(93.55;\, 106.45)$ соответствует уровню доверия:

$$\boxed{\gamma \approx 0.986} \quad \text{или} \quad \boxed{\gamma \approx 98.6\%}$$

## 6. Вывод

Интервал $(93.55;\, 106.45)$ шире 95%-ного, поэтому ему соответствует более высокий уровень доверия — примерно 98.6%. Это логично: чем шире интервал, тем с большей вероятностью он накрывает истинное значение параметра.

Можно сказать иначе: аналитик, использующий интервал $(93.55;\, 106.45)$, рискует ошибиться лишь в 1.4% случаев (против 5% для 95%-ного интервала), но платит за это расширением интервала с 10.32 до 12.90 тыс. км.

---

## Альтернативное решение: прямой подсчёт через $\Phi$

Можно было не вычислять $z_{1-\alpha_1/2}$ через пропорцию, а сразу заметить, что полуширина $\Delta = z_{1-\alpha/2} \cdot \frac{\sigma}{\sqrt{n}}$, и из того, что $\Delta_{0.95} = 5.16$ соответствует $z = 1.96$, найти:

$$\frac{\sigma}{\sqrt{n}} = \frac{5.16}{1.96} \approx 2.6327$$

Тогда для $I_1$:
$$z_{1-\alpha_1/2} = \frac{6.45}{2.6327} = 2.45$$

$$\gamma_1 = 2\Phi(2.45) - 1 = 2 \cdot 0.9929 - 1 = 0.9858$$

Результат тот же.

## Проверочный код Java

```java
public class Task4_4 {
    public static void main(String[] args) {
        // Из условия
        double mid95 = (94.84 + 105.16) / 2;
        double mid1 = (93.55 + 106.45) / 2;
        double half95 = (105.16 - 94.84) / 2;
        double half1 = (106.45 - 93.55) / 2;

        System.out.printf("Среднее x_bar = %.2f%n", mid95);
        System.out.printf("Полуширина 95%% интервала: %.2f%n", half95);
        System.out.printf("Полуширина искомого интервала: %.2f%n", half1);

        double zRatio = half1 / half95;
        double z1 = 1.96 * zRatio;
        System.out.printf("z-квантиль искомого интервала: %.2f%n", z1);

        // Вычисление Ф(2.45) через аппроксимацию
        double phi = cdfNormal(z1);
        double gamma = 2 * phi - 1;
        System.out.printf("Уровень доверия: %.4f (%.1f%%)%n", gamma, gamma * 100);
    }

    /** CDF стандартного нормального распределения (аппроксимация) */
    static double cdfNormal(double x) {
        return 0.5 * (1 + erf(x / Math.sqrt(2)));
    }

    /** Аппроксимация функции ошибок (Abramowitz and Stegun) */
    static double erf(double x) {
        // https://en.wikipedia.org/wiki/Error_function#Numerical_approximations
        double t = 1.0 / (1.0 + 0.3275911 * Math.abs(x));
        double tau = t * (0.254829592
                        + t * (-0.284496736
                        + t * (1.421413741
                        + t * (-1.453152027
                        + t * 1.061405429))));
        double y = 1 - tau * Math.exp(-x * x);
        return x >= 0 ? y : -y;
    }
}
```
