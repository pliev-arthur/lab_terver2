# Задача 9

## Условие

С помощью метода максимального правдоподобия оценить неизвестный(-е) параметр(-ы) гипотетического распределения по набору данных.

**Гипотетическое распределение:** $F_1: X \sim \mathcal{N}(a, \sigma^2)$ — нормальное распределение.

---

## 1. Обозначения

- $X \sim \mathcal{N}(a, \sigma^2)$ — нормальное распределение
- $a$ — математическое ожидание (неизвестный параметр)
- $\sigma^2$ — дисперсия (неизвестный параметр)
- $x_1, x_2, ..., x_n$ — выборка объёма $n = 200$
- $L(a, \sigma^2)$ — функция правдоподобия
- $\ln L(a, \sigma^2)$ — логарифмическая функция правдоподобия

## 2. Предположения

- Наблюдения независимы и одинаково распределены (i.i.d.)
- Распределение нормальное: $f(x) = \frac{1}{\sigma\sqrt{2\pi}} e^{-\frac{(x-a)^2}{2\sigma^2}}$

## 3. Вывод ММП-оценок

### 3.1. Функция правдоподобия

$$L(a, \sigma^2) = \prod_{i=1}^{n} \frac{1}{\sigma\sqrt{2\pi}} \exp\left(-\frac{(x_i - a)^2}{2\sigma^2}\right)$$

$$= (2\pi\sigma^2)^{-n/2} \exp\left(-\frac{1}{2\sigma^2} \sum_{i=1}^{n} (x_i - a)^2\right)$$

### 3.2. Логарифмическая функция правдоподобия

$$\ln L(a, \sigma^2) = -\frac{n}{2} \ln(2\pi) - \frac{n}{2} \ln(\sigma^2) - \frac{1}{2\sigma^2} \sum_{i=1}^{n} (x_i - a)^2$$

### 3.3. Уравнения правдоподобия

**Для параметра $a$** (приравниваем частную производную к нулю):

$$\frac{\partial \ln L}{\partial a} = \frac{1}{\sigma^2} \sum_{i=1}^{n} (x_i - a) = 0$$

$$\sum_{i=1}^{n} x_i - n a = 0$$

$$\boxed{\hat{a}_{ML} = \frac{1}{n} \sum_{i=1}^{n} X_i = \bar{X}}$$

**Для параметра $\sigma^2$** (приравниваем частную производную к нулю):

$$\frac{\partial \ln L}{\partial \sigma^2} = -\frac{n}{2\sigma^2} + \frac{1}{2\sigma^4} \sum_{i=1}^{n} (x_i - a)^2 = 0$$

$$-n\sigma^2 + \sum_{i=1}^{n} (x_i - a)^2 = 0$$

Подставляем $\hat{a}_{ML} = \bar{X}$:

$$\boxed{\hat{\sigma}^2_{ML} = \frac{1}{n} \sum_{i=1}^{n} (x_i - \bar{X})^2 = S^2}$$

### 3.4. Проверка на максимум

Вторые производные отрицательны в найденной точке, что подтверждает, что это максимум, а не минимум.

Важно: ММП-оценка дисперсии $\hat{\sigma}^2_{ML}$ является **смещённой**: $M(\hat{\sigma}^2_{ML}) = \frac{n-1}{n} \sigma^2 \neq \sigma^2$. Несмещённая оценка: $s^2 = \frac{n}{n-1} \hat{\sigma}^2_{ML}$.

## 4. Расчёты

### 4.1. ММП-оценка $a$

$$\hat{a}_{ML} = \bar{X} = \frac{1}{200} \sum_{i=1}^{200} x_i$$

Точное значение (вычислено в задаче 7): $\hat{a}_{ML} = -4.1621$

### 4.2. ММП-оценка $\sigma^2$

$$\hat{\sigma}^2_{ML} = \frac{1}{200} \sum_{i=1}^{200} (x_i - \bar{X})^2$$

Сначала вычислим сумму квадратов отклонений:

Используя ранее вычисленное значение дисперсии $S^2 = 27.6927$ (задача 7):

$$\sum_{i=1}^{200} (X_i - \bar{X})^2 = n \cdot S^2 = 200 \cdot 27.6927 = 5538.54$$

$$\hat{\sigma}^2_{ML} = \frac{5538.53}{200} = 27.6926$$

### 4.3. ММП-оценка $\sigma$

$$\hat{\sigma}_{ML} = \sqrt{\hat{\sigma}^2_{ML}} = \sqrt{27.6926} = 5.2624$$

## 5. Свойства полученных оценок

| Свойство | $\hat{a}_{ML}$ | $\hat{\sigma}^2_{ML}$ |
|:---|:---:|:---:|
| Несмещённость | Да | Нет ($M(\hat{\sigma}^2_{ML}) = \frac{199}{200}\sigma^2$) |
| Состоятельность | Да | Да |
| Эффективность | Да (достигает границы Крамера-Рао) | Асимптотически да |
| Достаточность | Да | Да (совместно с $\hat{a}_{ML}$) |

Для практических целей иногда используют исправленную (несмещённую) оценку: $\tilde{\sigma}^2 = \frac{n}{n-1} \hat{\sigma}^2_{ML} = \frac{200}{199} \cdot 27.6926 = 27.8318$, что обозначается $s^2$.

## 6. Результат

$$\boxed{\hat{a}_{ML} = -4.162}$$

$$\boxed{\hat{\sigma}^2_{ML} = 27.693 \quad (\hat{\sigma}_{ML} = 5.262)}$$

Оценки максимального правдоподобия для параметров нормального распределения по выборке столбца 5.

---

## Альтернативное решение: метод моментов

Метод моментов для нормального распределения даёт те же оценки, что и ММП (для нормального распределения ММП и метод моментов совпадают):

$$\hat{a}_{MM} = \bar{X} = -4.162$$
$$\hat{\sigma}^2_{MM} = \frac{1}{n}\sum (X_i - \bar{X})^2 = 27.693$$

Совпадение оценок — характерное свойство нормального распределения (для большинства других распределений ММП и метод моментов дают разные результаты).
